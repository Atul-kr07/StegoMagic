How to Use

1️⃣ Click "Choose Image" to upload an image.

2️⃣ Select either "Encode" or "Decode".

3️⃣ If encoding, enter a message and a security key.

4️⃣ Click "Encode" and download the encoded image.

5️⃣ If decoding, upload an encoded image and enter the correct key.

6️⃣ Click "Decode" to retrieve the hidden message.

7️⃣ Click "Reset" to go back to the start and choose a new function (Encode/Decode).
